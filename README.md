# DJE Scraper Worker v11.0 - DJEN

Worker para scraping de publicações judiciais do TJSP usando o sistema DJEN (Diário Nacional do CNJ).

## ⚠️ IMPORTANTE

O sistema DJE TJSP (dje.tjsp.jus.br) foi **descontinuado em 22/07/2025**.

Este worker agora acessa:
1. **DJEN** (comunica.pje.jus.br) - Sistema Nacional do CNJ
2. **DataJUD API** - API oficial do CNJ (fallback)
